# Geografia
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]
## Archivos json de Geografía
* [[1](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/1%20Paises.json)] Paises con código ISO y prefijo telefónico del país
 * [[2](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/2%20timezones.json)] Lista de husos horarios (GMT/UTC) y ciudades asociadas.
 * [[3](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/3%20provincias_ar.json)] Lista de provincias de Argentina con su código postal.
 * [[4](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/4%20latam_currencies.json)] Nombres y códigos de las monedas de América Latina.
 * [[5](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/5%20fronteras.json)] Países y sus países vecinos colindantes.
 * [[6](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/6%20climas_mundo.json)] Clasificación climática por región.
 * [[7](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/7%20coordenadas_ciudades.json)] Latitud y longitud de las principales ciudades.
 * [[8](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/8%20idiomas_oficiales.json)] Lista de países y sus idiomas oficiales.
 * [[9](https://github.com/webmastervetea/Json/blob/main/Geograf%C3%ADa/9%20oceanos.json)] Datos sobre oceanos (profundidad, ubicación).
