# Json
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]
## Arte y Creatividad

* [[1](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/1-paletas-colores.json)] Colecciones de códigos de colores (HEX/RGB) para diseño.
* [[2](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/2-tipografias-web.json)] Lista de fuentes con sus URLs y pesos (weights).
* [[3](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/3-poemas-clasicos.json)] Título y contenido de poemas organizados por autor.
* [[4](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/4-artistas-renacimiento.json)] Biografías cortas y obras clave de artistas.
* [[5](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/5-presets-filtros.json)] Parámetros para filtros de imágenes (brillo, contraste, saturación).
* [[6](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/6-sonidos-interfaz.json)] Nombres de archivos de sonido para notificaciones (alerta, error, éxito).
* [[7](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/7-recetas-cocina.json)] Ingredientes y pasos para platos específicos.
* [[8](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/8-instrumentos-musicales.json)] Clasificación de instrumentos por familia (cuerda, viento, percusión).
* [[9](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/9-comics-personajes.json)] Nombres de personajes de historietas y sus superpoderes.
* [[10](https://github.com/webmastervetea/Json/blob/main/arte-y-creatividad/10-patrones-geometricos.json)] Coordenadas para dibujar formas y patrones complejos.
