# Json
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]

## Desarrollo de Software y Servicios Web [[Más categorías](https://github.com/webmastervetea/Json/blob/main/CATEGORIAS.md)]

* [[1](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/1-test-data_users.json)] Datos ficticios de usuarios para pruebas unitarias/de integración. 
* [[2](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/2-mock-api-response.json)] Respuesta simulada de un servicio externo para desarrollo *offline*. 
* [[3](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/3-database-schema.json)] Definición de las tablas, campos y relaciones de una DB. 
* [[4](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/4-log-levels.json)] Niveles de detalle para el registro de eventos (*logging*). 
* [[5](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/5-cron-jobs.json)] Programación de tareas automáticas y recurrentes.
* [[6](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/6-cdn-settings.json)] Configuración para la Red de Distribución de Contenidos (CDN). 
* [[7](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/7-middleware-chain.json)] Orden y configuración de las funciones de *middleware*. 
* [[8](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/8-webhooks.json)] URLs y eventos para notificaciones de webhooks. 
* [[9](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/9-oaut-clients.json)] IDs y secretos de clientes para autenticación OAuth. 
* [[10](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/10-rate-limits.json)] Límites de solicitudes por usuario o IP para una API. 
* [[11](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/11-tooltips.json)] Textos de ayuda que aparecen al pasar el ratón sobre elementos. 
* [[12](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/12-onboarding-steps.json)] Contenido y orden de los pasos de bienvenida a nuevos usuarios.
* [[13](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/13-breakpoints.json)] Tamaños de pantalla para diseño web *responsive*.
* [[14](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/14-icon-map.json)] Mapeo de nombres de iconos a sus archivos de imagen (SVG). 
* [[15](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/15-enum-status.json)] Enumeración de estados posibles (activo, inactivo, pendiente). 
* [[16](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/16-error-codes.json)] Códigos de error estandarizados y mensajes para el usuario. 
* [[17](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/17-default-user-settings.json)] Preferencias iniciales para una cuenta nueva. 
* [[18](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/18-validation-rules.json)] Reglas para validar la entrada de datos en formularios. 
* [[19](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/19-social-media-links.json)] URLs a las redes sociales de la empresa o persona. 
* [[20](https://github.com/webmastervetea/Json/blob/main/desarrollo-software-y-servicios-web/20-faq-sections.json)] Preguntas frecuentes organizadas por categoría.
