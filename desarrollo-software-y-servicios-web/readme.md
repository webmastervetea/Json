# Json
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]

## Desarrollo de Software y Servicios Web [[Más categorías](https://github.com/webmastervetea/Json/blob/main/CATEGORIAS.md)]

* [[1]()] Datos ficticios de usuarios para pruebas unitarias/de integración. 
* [[2]()] Respuesta simulada de un servicio externo para desarrollo *offline*. 
* [[3]()] Definición de las tablas, campos y relaciones de una DB. 
* [[4]()] Niveles de detalle para el registro de eventos (*logging*). 
* [[5]()] Programación de tareas automáticas y recurrentes.
* [[6]()] Configuración para la Red de Distribución de Contenidos (CDN). 
* [[7]()] Orden y configuración de las funciones de *middleware*. 
* [[8]()] URLs y eventos para notificaciones de webhooks. 
* [[9]()] IDs y secretos de clientes para autenticación OAuth. 
* [[10]()] Límites de solicitudes por usuario o IP para una API. 
* [[11]()] Textos de ayuda que aparecen al pasar el ratón sobre elementos. 
* [[12]()] Contenido y orden de los pasos de bienvenida a nuevos usuarios.
* [[13]()] Tamaños de pantalla para diseño web *responsive*.
* [[14]()] Mapeo de nombres de iconos a sus archivos de imagen (SVG). 
* [[15]()] Enumeración de estados posibles (activo, inactivo, pendiente). 
* [[16]()] Códigos de error estandarizados y mensajes para el usuario. 
* [[17]()] Preferencias iniciales para una cuenta nueva. 
* [[18]()] Reglas para validar la entrada de datos en formularios. 
* [[19]()] URLs a las redes sociales de la empresa o persona. 
* [[20]()] Preguntas frecuentes organizadas por categoría.
