# Json
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]

## Juegos y Entretenimiento [[Más categorías](https://github.com/webmastervetea/Json/blob/main/CATEGORIAS.md)]
* [[1](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/1-game-levels.json)] Configuración y diseño de niveles para un videojuego.
* [[3](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/3-high-scores.json)] Puntuaciones más altas y nombres de jugadores.
* [[4](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/4-dialogues.json)] Scripts de diálogo para personajes de un juego o novela visual.
* [[5](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/5-pokemon-stats.json)] Nombres, tipos y estadísticas base de criaturas (e.g., Pokémon).
* [[6](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/6-recetas-crafting.json)] Combinaciones de ítems para crear nuevos objetos.
* [[7](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/7-peliculas-favoritas.json)] Títulos de películas, año y puntuación personal.
* [[8](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/8-lista-musica-generos.json)] Clasificación de álbumes o canciones por género.
* [[9](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/9-trivia-questions.json)] Preguntas y respuestas para un juego de trivia.
* [[10](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/10-e-sports-teams.json)] Nombre de equipos profesionales de eSports y sus jugadores.
* [[11](https://github.com/webmastervetea/Json/blob/main/juegos-y-entretenimiento/11-inventory.json)] Inventaria de juegos.
