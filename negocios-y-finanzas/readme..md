# Json
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]

## Negocios y Finanzas [[Más categorías](https://github.com/webmastervetea/Json/blob/main/CATEGORIAS.md)]

* [[1](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/1-stock-prices.json)] Precios de cierre y apertura de acciones en un día específico.
* [[2](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/2-currencies-exchange.json)] Tasas de cambio entre diferentes divisas.
* [[3](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/3-empleados-data.json)] Información básica de la plantilla (nombre, departamento, ID).
* [[4](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/4-productos-catalogo.json)] Artículos de venta (nombre, SKU, precio, stock).
* [[5](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/5-invoices-pending.json)] Facturas pendientes de pago (ID, cliente, monto).
* [[6](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/6-clientes-premium.json)] Lista de clientes VIP o con planes especiales.
* [[7](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/7-gastos-departamento.json)] Resumen de gastos por área o departamento.
* [[8](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/8-criterios-crediticios.json)] Reglas y puntajes para la aprobación de crédito.
* [[9](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/9-calendario-fiscal.json)] Fechas importantes de presentación de impuestos.
* [[10](https://github.com/webmastervetea/Json/blob/main/negocios-y-finanzas/10-metodos-pago.json)] Tipos de pago aceptados y sus configuraciones.
