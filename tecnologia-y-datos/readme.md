# Json
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]

## Tecnología y Datos [[Más categorías](https://github.com/webmastervetea/Json/blob/main/CATEGORIAS.md)]

* [[1](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/1-data-schema.json)] Definición de la estructura y tipos de datos esperados en una base de datos.
* [[2](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/2-logs-errores.json)] Registro de fallos de un sistema (timestamp, código de error, mensaje).
* [[3](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/3-hardware-specs.json)] Especificaciones de componentes de computadora (CPU, GPU, RAM).
* [[4](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/4-protocolos-red.json)] Lista de protocolos de red y sus puertos (HTTP, FTP, SSH).
* [[5](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/5-ml-parameters.json)] Hiperparámetros para un modelo de machine learning.
* [[6](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/6-diccionario-terminos.json)] Glosario de términos técnicos con su definición.
* [[7](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/7-updates-history.json)] Historial de versiones y cambios (changelog) de un software.
* [[8](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/8-backup-status.json)] Estado de las últimas copias de seguridad.
* [[9](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/9-sensores-iot.json)] Datos capturados de dispositivos IoT (temperatura, humedad).
* [[10](https://github.com/webmastervetea/Json/blob/main/tecnologia-y-datos/10-crypto-wallets.json)] Registros de direcciones de monederos de criptomonedas.
