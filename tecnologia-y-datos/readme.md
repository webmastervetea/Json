# Json
Este repositorio es una colección organizada de pequeñas archivos en formato .json que pueden ser utilizados por programadores en sus desarrollos.
[[Soporte](https://www.linkedin.com/in/oscarlizarragag/)]

## Tecnología y Datos [[Más categorías](https://github.com/webmastervetea/Json/blob/main/CATEGORIAS.md)]

* 1 Definición de la estructura y tipos de datos esperados en una base de datos.
* 2 Registro de fallos de un sistema (timestamp, código de error, mensaje).
* 3 Especificaciones de componentes de computadora (CPU, GPU, RAM).
* 4 Lista de protocolos de red y sus puertos (HTTP, FTP, SSH).
* 5 Hiperparámetros para un modelo de machine learning.
* 6 Glosario de términos técnicos con su definición.
* 7 Historial de versiones y cambios (changelog) de un software.
* 8 Estado de las últimas copias de seguridad.
* 9 Datos capturados de dispositivos IoT (temperatura, humedad).
* 10 Registros de direcciones de monederos de criptomonedas.
